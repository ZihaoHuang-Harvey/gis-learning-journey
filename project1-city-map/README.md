# 北京景点地图

基于 OpenLayers 的 WebGIS 可视化项目，展示北京主要景点分布。

## 在线预览

[点击访问](https://zihaohuang-harvey.github.io/gis-learning-journey/project1-city-map/index.html)

## 功能特性

- 🗺️ 三种底图切换（街道/卫星/地形）
- 📍 10个北京景点标记（历史/自然/现代分类）
- 🔍 景点搜索，地图自动定位
- 🏷️ 分类筛选（全部/历史文化/自然风光/现代地标）
- 📱 响应式布局，支持手机访问
- 🖱️ 点击标记显示景点详情弹窗
- 📐 比例尺、全屏、缩放滑块、鼠标坐标显示

## 技术栈

- OpenLayers 8.2.0
- HTML5 / CSS3 / JavaScript (ES6)
- GeoJSON 数据格式
